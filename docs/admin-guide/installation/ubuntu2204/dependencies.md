# Install Dependencies

## Instructions

### Step 1. Connect to Virtual Machine
```shell
ssh <username>@<server-ip>
```

### Step 2. Update APT 
```shell
sudo apt update
```

### Step 3. Install Dependencies

#### Install Java
```shell
sudo apt install openjdk-8-jre-headless
```

#### Install Tomcat
```shell
sudo apt install tomcat9
```

#### Install MySQL
```shell
sudo apt install mysql-server
```
